## 📋 项目简介

「实习能量站」是一套面向业务部门的 **实习生成长导航智能看板**，采用 **三角色联动（实习生 · 导师 · HR）+ AI 数据穿透** 的设计架构，解决传统实习管理中「带教凭感觉、成长无可视、转正靠主观」三大痛点。

## 🎯 核心亮点 · 对标五维盲评

| 评分维度 | 核心能力 |
|:---:|---|
| **思辨深度** | 🧠 AI 成长洞察自动分析五维优劣 → 风险预警 AI 干预（根因分析 + 时间线方案）→ 全员去留报告一键生成 |
| **创意巧思** | 🔄 实习生选择器自由切换 20 人视角 → 任务状态点击即改实时反馈 → 评估滑块联动雷达图 |
| **功能完整度** | 📦 12 个功能视图全实现 + 6 个 AI 功能按钮 + 交互操作 |
| **交互体验** | ✨ 浮动光球登录背景 → 数字跳动动画 → 角色切换脉冲光晕 → 深色玻璃拟态主题 |
| **落地可行性** | 🚀 纯前端零后端 → SiliconFlow API → GitHub Pages 一键部署 → 可平替腾讯混元 |

## 🖥️ 功能全景

### 实习生视角 👨‍🎓
| 模块 | 功能说明 |
|---|---|
| 🗺️ 我的成长地图 | 30-60-90 天里程碑 + AI 成长洞察 + 评分趋势折线图 |

| 📝 成长周报 | AI 一键生成结构化周报 + 历史评分趋势 |

### 导师视角 👨‍🏫
| 模块 | 功能说明 |
|---|---|
| 📌 带教看板 | 学员概览表 + 能力对比雷达图 |

| 🚀 转正预测 | AI 综合排名 + **一键生成全员去留分析报告** |

## 🤖 AI 集成能力

## 🛠️ 技术栈

```
前端框架：    原生 HTML + CSS + JavaScript（零框架依赖）
设计系统：    深色玻璃拟态（Glassmorphism）· 4200+ 行 CSS
数据可视化：  Chart.js 4.4（雷达图/折线图/柱状图/饼图/九宫格）
AI 引擎：     SiliconFlow API · Qwen2.5-7B-Instruct
字体：        Outfit + Noto Sans SC（Google Fonts）
部署：        GitHub Pages 静态部署
```
