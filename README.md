## 💡 项目背景

传统业务部在校招实习生管理中存在三大痛点：

- 🔴 **带教凭感觉** — 导师缺乏结构化带教工具，标准难落地
- 🟡 **成长无可视** — 学习进度、能力评估散落在周报、IM、Excel 中  
- 🔵 **转正靠主观** — HR 缺乏跨部门横向对比数据，无法量化决策

「实习能量站」采用 **三角色联动 + AI 数据穿透** 的设计架构，让实习生管理从「凭感觉」走向「看数据」。
